# TMS Spring Educators #EdTech Conference 2026

**Tech My School · April 10–11, 2026 · Wyndham Rio Mar, Puerto Rico**

Conference web app for the 2026 Spring Educators EdTech Conference.

## Features
- Conference home page with speaker directory
- Session browser (Day 1 & Day 2)
- Attendee registration & check-in
- Session selection flow
- Interactive venue floor map
- Admin portal (badge queue, analytics, CSV export)

Live at: https://tms-conference.vercel.app
